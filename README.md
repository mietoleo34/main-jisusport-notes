# main-jisusport-notes

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为 HTML 文件的存储与展示空间，方便用户浏览、查阅或引用其中的页面内容。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库主页（可选）
├── pages/             # 存放独立 HTML 页面
│   └── ...
└── assets/            # 公共资源（如样式、图片、脚本）
    └── ...
```

- `pages/` 目录下存放各个独立的 HTML 页面文件，每个页面可包含自身所需的样式与脚本。
- `assets/` 目录用于存放多个页面可能共享的资源文件，便于统一管理。

## 页面归档说明

本仓库中的 HTML 页面均以独立文件形式存在，每个页面可视为一份独立的文档或工具。页面内容可能涵盖笔记、示例、演示或其它非商业性用途。所有页面均不嵌入外部不可控资源，避免依赖第三方服务。

如需新增页面，请在 `pages/` 下创建新的 HTML 文件，并在必要时引用 `assets/` 中的公共资源。

## 使用方式

你可以通过以下方式浏览仓库中的 HTML 页面：

1. **直接查看源码**：在仓库中点击任意 `.html` 文件，GitHub 会自动渲染并展示页面内容。
2. **克隆到本地**：使用 `git clone` 将仓库下载到本地，用浏览器打开对应 HTML 文件即可。
3. **通过 GitHub Pages 部署**（如已启用）：访问仓库的 Pages 链接，可直接在线浏览所有页面。
